# 2026-adhoc-bmi_visualizer

身長・性別・年齢を入力すると、次の 2 種類の情報を並べて確認できる静的 HTML ツールです。

- 厚生労働省基準の標準体重
- e-Stat 平均値に基づく参考的な平均体重・平均 BMI

単一の [index.html](index.html) で動作するため、そのまま GitHub Pages で公開しています。

## 概要

このツールは、同じ入力値に対して性質の異なる 2 つの見方を表示します。

1. 厚生労働省の情報に基づいた標準体重
2. e-Stat の平均値に基づいた参考表示

上段は BMI 22 を基準にした標準体重で、計算に使うのは身長のみです。性別と年齢はこの段の計算には使用しません。

下段は e-Stat の年齢階級別平均値を元にした参考表示です。性別と年齢階級に対応する平均 BMI を用い、入力された身長に当てはめた参考的な平均体重を算出します。

## 使い方

1. [index.html](index.html) をブラウザで開きます。
2. 身長、性別、年齢を入力します。
3. 計算する ボタンを押します。
4. 上段で標準体重、下段で参考的な平均体重と平均 BMI を確認します。

## 表示内容

### 厚生労働省基準の標準体重

- 成人の標準指標である BMI 22 を基準に算出します。
- 健康体重の範囲は BMI 18.5〜25.0 を元に表示します。
- バー下には目安として 20歳〜64歳 と 65歳以上 の BMI 範囲を文字列で表示します。
- この段の計算には身長のみを使用します。

### e-Stat の平均値に基づいた参考表示

- 2019年 国民健康・栄養調査の年齢階級別平均値を使用します。
- 参照年齢階級、平均身長、平均体重、平均 BMI を表示します。
- 平均 BMI を入力身長へ当てはめた 参考的な平均体重 を表示します。
- BMI バーは 65歳未満 と 65歳以上 で目安帯を切り替えて表示します。

## データ出典

### 厚生労働省基準

- 厚生労働省 e-健康づくりネット「肥満と健康」
	- https://kennet.mhlw.go.jp/information/information/food/e-02-001

### e-Stat 平均値

- e-Stat 政府統計の総合窓口
- 国民健康・栄養調査 2019年 身長・体重の平均値及び標準偏差
- 元データ CSV: [resource/FEH_00450171_260531202449.csv](resource/FEH_00450171_260531202449.csv)

## 注意事項

- 本ツールは参考情報の提供を目的としており、医療上のアドバイスに代わるものではありません。
- e-Stat 側は平均値ベースの参考表示です。骨量、筋肉量、体脂肪率、生活習慣などにより適正な体格は大きく異なります。
